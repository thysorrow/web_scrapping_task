Для запуска нужен python 3.13.3

Для установки зависимостей: скопировать корневую папку, находясь в ней в терминале вызвавть команду 
```pip install - requirements.txt```

Для запуска тестов:
Находясь в корневой директории запустить команду в терминале
```pytest```

С помощью флага ```-v``` можно увидеть более детальный отчет по тестам

Пример запуска конкретного файла теста
```pytest test_link_extractor.py```

Пример запуска конкретного теста в файле
```pytest test_link_extractor.py::test_extract_links```

Для просмотра информации по покрытию кода тестами:
```pytest --cov```

Пример запуска теста с подробной информацией и с информации о покрытии кода:
```pytest test_summary_formatter.py -v --cov```
