Для запуска нужен python 3.13.3

Для установки зависимостей: находясь в корневой директории в терминале вызвать команду 
```pip install -r requirements.txt```

Для запуска тестов:
Находясь в корневой директории запустить команду в терминале
```pytest```

С помощью флага ```-v``` можно увидеть более детальный отчет по тестам

Пример команды для запуска конкретного файла теста:
```pytest test_link_extractor.py```

Пример команды для запуска конкретного теста в файле:
```pytest test_link_extractor.py::test_extract_links```

Для просмотра информации по покрытию кода тестами:
```pytest --cov```

Пример команды для запуска теста с подробной информацией и с информации о покрытии кода:
```pytest test_summary_formatter.py -v --cov```
